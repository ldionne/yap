Macro BOOST_YAP_CONVERSION_OPERATOR_TEMPLATE

BOOST_YAP_CONVERSION_OPERATOR_TEMPLATE

Synopsis

// In header: <boost/yap/config.hpp>

BOOST_YAP_CONVERSION_OPERATOR_TEMPLATE

Description

Controls whether a conversion operator template is defined for expression<>.

Default value is 0.

This is useful when you want to have expression<>s auto-evaluated on assignment to non-yap types. It can also wreak havok on your code due to unintentional implicit conversions. Here be dragons.
