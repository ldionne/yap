Function template evaluate

boost::yap::evaluate

Synopsis

// In header: <boost/yap/expression.hpp>


template<typename Expr, typename... T> 
  decltype(auto) evaluate(Expr && expr, T &&... t);

Description

Evaluates expr, substituting the subsequent parameters (if any) into expr's placeholders.

All customization points for the evaluation of expressions except eval_expression_as() are used to evaluate the expr. If you've overridden any, that will be reflected in the result.

Note

evaluate() is only valid if Expr is an expression, and max_p <= sizeof...(T), where max_p is the maximum placeholder index in expr.
